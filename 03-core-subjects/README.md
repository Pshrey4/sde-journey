# 📚 Core Computer Science Subjects

> *"Strong software engineers don't just know how to code—they understand how computers work underneath."*

This folder contains my notes, learning resources, and revision material for the core Computer Science subjects required for software engineering interviews and real-world software development.

My focus is on building a solid understanding of system fundamentals before moving on to advanced topics like System Design and Backend Development.

---

## 🎯 Subjects Covered

### 💻 Operating Systems (OS)
Learn how an operating system manages processes, memory, files, CPU scheduling, synchronization, deadlocks, virtual memory, and more.

### 🗄️ Database Management Systems (DBMS)
Study relational databases, SQL, normalization, transactions, indexing, concurrency control, and database internals.

### 🌐 Computer Networks (CN)
Understand networking fundamentals including the OSI & TCP/IP models, HTTP/HTTPS, DNS, routing, congestion control, transport protocols, and network security.
