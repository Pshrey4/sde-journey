# Conditional Statement Notes  

**1. if...else**
```  
if(condition){  
    //condition is true then run this block of code  
}  
else {  
    //condition is false ruun this block of code  
}  
```

**Note: False - 0, True - 1 (anything else other than 0)**  

**Nested if**  
```
if(condition){  
    if(condition){  
        //code    
    }  
    else{  
        //code    
    }  
}  
else{  
    if(condition){  
        //code    
    }  
    else{  
        //code    
    }  
}  
```
